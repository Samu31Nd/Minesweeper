# Minesweeper
**Made in JavaFX**\
With some css style aplied.

### About the game
The game has 3 gamemodes:
- #### Easy:
  - **Board:** 9x9.
  - **Number of mines:** 10.
- #### Medium:
  - **Board:** 16x16.
  - **Number of mines:** 40.
- #### Hard:
  - **Board:** 16x30.
  - **Number of mines:** 99.
